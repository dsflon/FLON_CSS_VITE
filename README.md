# FLONCSS TEMPLATE FOR VITE

node 18.x 以上

# Setup & Start

```bash
$ npm install

$ npm run dev
```

http://localhost:3000/ で画面確認できます。  

### 本番公開用 build
```
npm run build
```
distフォルダにbuildファイルが生成されます。  
本番公開する場合は必ず build したものをアップロードしてください。
